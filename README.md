# Aiden Portfolio Site

Aiden의 커리어 전환 스토리와 AI 기반 제품 제작 역량을 보여주기 위한 한국어 포트폴리오 사이트입니다.

## 포함 섹션

- 랜딩(히어로)
- About
- Projects (TOEIC Daily, JLPT Daily, TOPIK Daily, Solo Survival)
- Strengths (기획력, AI 활용, 실행력, 운영 경험)
- Contact (플레이스홀더)

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 표시되는 로컬 주소(기본값 `http://localhost:5173`)로 접속하세요.

## 빌드

```bash
npm run build
npm run preview
```
