# Aiden's Portfolio

**Live Site:** https://portfolio-site-sage-six.vercel.app

에이든의 포트폴리오 사이트입니다.  
전략기획/사업개발 관점의 문제를 AI 기반 제품으로 구조화해 보여주기 위한 개인 포트폴리오로, 핵심 전략 데모 3종과 소개/강점/연락처 정보를 담고 있습니다.

## What this site shows

- **About** — 에이든의 배경과 문제 해결 방식
- **Strategy Projects** — 아래 3개 데모 프로젝트 소개 및 링크
  - Market & Competitor Intelligence Dashboard
  - New Business Idea Evaluation Dashboard
  - Strategic Action Tracker
- **Strengths** — 전략 기획, AI 활용 제품화, 비즈니스 개발, 실행 중심 사고
- **Contact** — Email / LinkedIn / GitHub

## Key UX

- 한국어 / English 언어 전환 지원
- 지구본(🌐) 기반 언어 선택 UI
- 각 프로젝트별 **Live Demo / GitHub** 링크 제공
- 반응형 단일 페이지 포트폴리오

## Tech Stack

- React
- Vite
- Custom CSS
